## Overview

Produce a formated form from definitions and array of values.

## Call

`form::call(['name'=>'var','age'=>'int'],['name'=>'bob','age'=>'18']);`

## Functions

- `tabler($r)`
- `build($r)` -- used in : `form::call`
- `call($ra,$r=[])` -- used in : `edit::create`, `edit::edit`, `nav::create`
